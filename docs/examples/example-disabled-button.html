Disabled button

In scenarions where you don't want to allow users to submit before all validations are met, you may want to add disabled attribute to your submit button or add submit option to true.

Your button will be automatically enabled once all the form fields meet their validation rules.

CSS

This form is using a custom stylesheet build upon forma-boilerplate.css. Take a look at the code on the GitHub repo.

HTML

<form class="forma">
  <label>
    <span>Username</span>
    <div class="forma-fieldset">
      <input type="text" required />
    </div>
  </label>
  <label>
    <span>Password</span>
    <div class="forma-fieldset">
      <input type="password" required />
    </div>
  </label>
  <button type="submit">Submit</button>
</form>

JavaScript


forma({
  submit: true
});
