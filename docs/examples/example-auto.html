auto:true

Sample form with auto option set to true. This will automatically add and generate description and placeholder to each field.

HTML

<form class="forma">
    <label>
        <span>Username</span>
        <div class="fieldset">
            <input type="text" />
        </div>
    </label>
    <label>
        <span>Password</span>
        <div class="fieldset">
            <input type="password" />
        </div>
    </label>
    <button type="submit">Submit</button>
</form>

JavaScript

$.forma({
    auto: true
});
