Disabled button

In scenarions where you don't want to allow users to submit before all validation are met, you may want to set the disabled attribute to your submit button.

Your button will be automatically enabled once all the form fields meet their validation rules.

HTML

<form class="forma">
    <label>
        <span>Username</span>
        <div class="fieldset">
            <input type="text" required />
        </div>
    </label>
    <label>
        <span>Password</span>
        <div class="fieldset">
            <input type="password" required />
        </div>
    </label>
    <button type="submit" disabled>Submit</button>
</form>

JavaScript

$.forma();
