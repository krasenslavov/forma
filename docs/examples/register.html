Register form (real-world example)

Generic real-world registration form.
Using forma extended CSS with which the user is prompt only one field at a time, moving to the next using the tab navigation. Once all the fields are complete with valid data the button is enabled for submission.

CSS

.forma-register label:not(.forma-open) {
    display: none;
}

HTML

<form class="forma forma-register">
    <label for="email">
        <span>Email address</span>
        <div class="fieldset">
            <input type="email" name="email" pattern="\S+@\S+\.\S+" 
            title="Enter a valid email address." required />
        </div>
    </label>
    <label for="fullname">
        <span>Full Name</span>
        <div class="fieldset">
            <input type="text" name="fullname" pattern="[a-zA-Z]{3,32}" 
            title="Full name must be between 3 and 32 characters." required />
        </div>
    </label>
    <label for="username">
        <span>Username</span>
        <div class="fieldset">
            <input type="text" name="username" pattern="[a-z]{3,32}" 
            title="Username must be lowercase between 3 and 32 characters." required />
        </div>
    </label>
    <label for="password">
        <span>Password</span>
        <div class="fieldset">
            <input type="password" name="password" pattern="(?=.*[A-Za-z])(?=.*\d)[A-Za-z\d]{8,}" 
            title="Password must have minimum 8 characters, at least one letter and one number" required />
        </div>
    </label>
    <label for="confirm-password">
        <span>Confirm password</span>
        <div class="fieldset">
            <input type="password" name="confirm-password" pattern="(?=.*[A-Za-z])(?=.*\d)[A-Za-z\d]{8,}" 
            title="Password must have minimum 8 characters, at least one letter and one number" required />
        </div>
    </label>
    <button type="submit" disabled>Register</button>
</form>

JavaScript

$.forma({
    container: '.forma-register',
    auto: true
});
