Multiple forms

Adding multiple forms on the same page is easily integrated.

All you need to do is to specify a unique class name for each target form and then initialize each form individually by its unique class.

Note: The tab navigation will loop only within the form and not go outside. .forma base class is required.



CSS

This form is using a custom stylesheet build upon forma-boilerplate.css. Take a look at the code on the GitHub repo.
