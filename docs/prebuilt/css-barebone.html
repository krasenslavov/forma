Barebone CSS

In additional to the basic structure some additional styles are added to this pre-built stylesheet.

Notices that now the tab label and manual toggler are visible (comparing it with the Base example).

CSS

This form is using a pre-built stylesheet forma-barebone.css. Take a look at the code on the GitHub repo.

HTML

<form class="forma">
  <label>
    <span>Username</span>
    <div class="forma-fieldset">
      <input type="text" required />
    </div>
  </label>
  <label>
    <span>Password</span>
    <div class="forma-fieldset">
      <input type="password" required />
    </div>
  </label>
  <button type="submit">Submit</button>
</form>

JavaScript

forma({
  auto: true,
  manual: true,
  show: true
});
