Login form (real-world example)

Generic real-world login form.

CSS

This form is using a custom stylesheet build upon forma-boilerplate.css. Take a look at the code on the GitHub repo.

HTML

<form class="forma forma-login">
  <label for="username">
    <span>Username</span>
    <div class="forma-fieldset">
      <input type="text" name="username" pattern="[a-z]{3,32}" 
      title="Username must be lowercase between 3 and 32 characters." required />
    </div>
  </label>
  <label for="password">
    <span>Password</span>
    <div class="forma-fieldset">
      <input type="password" name="password" pattern="(?=.*[A-Za-z])(?=.*\d)[A-Za-z\d]{8,}" 
      title="Password must have minimum 8 characters, at least one letter and one number." required />
    </div>
  </label>
  <button type="submit">Login</button>
</form>

JavaScript

forma({
  container: '.forma-login',
  auto: true,
  show: true,
  submit: true
});
