Multiple forms

Having multiple forms on the same page is very easy.
All you need to do is to specify a unique class name for each form tag and then initialize each form individually by its unique class.
Also, note that the the tab navigation will loop only within the form and not go outside.
* .forma base class is required.


HTML

<form class="forma forma-0">
    <label>
        <span>Username</span>
        <div class="fieldset">
            <input type="text" />
        </div>
    </label>
    <label>
        <span>Password</span>
        <div class="fieldset">
            <input type="password" />
        </div>
    </label>
    <button type="submit">Login</button>
</form>
<form class="forma forma-1">
     <label>
        <span>Email address</span>
        <div class="fieldset">
            <input type="email" />
        </div>
    </label>
    <label>
        <span>Full name</span>
        <div class="fieldset">
            <input type="text" />
        </div>
    </label>
    <label>
        <span>Username</span>
        <div class="fieldset">
            <input type="text" />
        </div>
    </label>
    <label>
        <span>Password</span>
        <div class="fieldset">
            <input type="password" />
        </div>
    </label>
    <label>
        <span>Confirm password</span>
        <div class="fieldset">
            <input type="password" />
        </div>
    </label>
    <button type="submit">Register</button>
</form>

JavaScript

$.forma({
    container: '.forma-0'
});
$.forma({
    container: '.forma-1',
    auto: true
});
